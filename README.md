# 🇫🇷 FRAGMENT ELITE

**Sécurité Souveraine & Fragmentation XOR locale.**

Fragment Elite est un outil open-source permettant de diviser un secret (Seed Phrase, mot de passe) en deux fragments mathématiques. Un seul fragment ne sert à rien, il faut les deux pour reconstituer le secret.

### ✨ Pourquoi l'utiliser ?
- **100% Local :** Aucune donnée ne quitte votre navigateur.
- **Zéro Serveur :** Fonctionne même hors-ligne.
- **Zéro Cloud :** Protection contre le piratage à distance.
